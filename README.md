# 📡 خادم البث المباشر المتكامل (Live Stream Server)

تطبيق ويب مبني باستخدام **Node.js** و **Express** و **Node-Media-Server** لاستقبال البث المباشر من برنامج **OBS Studio** وعرضه على المتصفحات بكفاءة عالية مع دعم كامل لأجهزة الحاسوب والهواتف الذكية (iOS & Android).

---

## ✨ المميزات

* ⚡ **تأخير منخفض جداً (Low Latency):** دعم بروتوكول HTTP-FLV على الحواسيب وأجهزة أندرويد لتقليل وقت التأخير إلى 1-2 ثانية فقط.
* 📱 **دعم الهواتف بالكامل (HLS):** توليد بث HLS متوافق تلقائياً مع هواتف iPhone (iOS) ومتصفح Safari.
* 🔍 **كشف ذكي وتلقائي:** يقوم الخادم بالتحقق من وجود أداة FFmpeg وتفعيل الميزات المتوافقة تلقائياً دون تدخل منك.
* 🖥️ **واجهة مستخدم عصرية:** تصميم داكن وتأثير زجاجي (Glassmorphism) مريح للعين ومتجاوب بالكامل مع كافة الشاشات.
* 🔗 **مشاركة سهلة:** يظهر لك الخادم عنوان الـ IP الخاص بجهازك تلقائياً لفتحه من هاتفك المحمول أو أي حاسوب آخر في نفس الشبكة المحلية.

---

## 🛠️ متطلبات التشغيل والتثبيت

### 1. تثبيت الحزم المطلوبة
إذا لم تقم بتثبيتها بعد، افتح المجلد في سطر الأوامر وشغل:
```bash
npm install
```
*(ملاحظة لمستخدمي Windows: إذا واجهت مشكلة صلاحيات، يمكنك التشغيل عبر: `powershell -ExecutionPolicy Bypass -Command "npm install"`)*

### 2. تثبيت FFmpeg (مهم جداً لدعم هواتف iPhone و Safari)
لكي يعمل البث على هواتف iPhone، يحتاج الخادم لأداة FFmpeg لتقسيم البث.
* **على نظام Windows (باستخدام PowerShell كمسؤول):**
  ```powershell
  winget install ffmpeg
  ```
  *بعد اكتمال التثبيت، يرجى إغلاق سطر الأوامر وفتحه مجدداً لإعادة تشغيل الخادم وتطبيق التغييرات.*

---

## 🚀 طريقة التشغيل والبدء

1. **شغل الخادم:**
   ```bash
   npm start
   ```
2. **افتح الموقع:**
   * على الحاسوب الشخصي: افتح المتصفح على [http://localhost:3000](http://localhost:3000)
   * على الهاتف المحمول (يجب أن يكون متصلاً بنفس شبكة الواي فاي): افتح الرابط الموضح في شاشة الخادم (مثال: `http://192.168.1.50:3000`).

---

## 🎥 ضبط برنامج OBS Studio للبث

1. افتح برنامج **OBS Studio**.
2. اذهب إلى الإعدادات (**Settings**) ثم البث (**Stream**).
3. اختر نوع الخدمة: مخصص (**Custom...**).
4. أدخل إعدادات الخادم (التي تظهر في صفحة الموقع):
   * **خادم البث (Server URL):** `rtmp://YOUR_IP_ADDRESS/live`
   * **مفتاح البث (Stream Key):** `mystream`
5. اضبط إعدادات المخرج (**Output**) للحصول على أفضل أداء وأقل تأخير:
   * **Output Mode:** Advanced
   * **Rate Control:** CBR
   * **Bitrate:** 2500 - 4500 Kbps (حسب سرعة الرفع لديك)
   * **Keyframe Interval:** 2 s
   * **Profile:** main أو high
   * **Tune:** zerolatency
6. اضغط على **Start Streaming** في OBS.
7. سيتحول الموقع تلقائياً إلى وضع **مباشر الآن** ويبدأ عرض البث!

## 🌐 البث خارج الشبكة المحلية (عبر الإنترنت)

إذا كنت ترغب في مشاركة البث مع أشخاص خارج منزلك أو شبكتك المحلية مجاناً، يمكنك استخدام أداة **ngrok** لإنشاء نفق آمن:

1. قم بتنزيل وتثبيت أداة [ngrok](https://ngrok.com/).
2. افتح سطر أوامر جديد وقم بتوجيه المنفذ 3000 (منفذ الواجهة):
   ```bash
   ngrok http 3000
   ```
3. سيعطيك ngrok رابطاً عاماً يبدأ بـ `https://xxxx.ngrok-free.app` يمكنك مشاركته مع أي شخص في العالم لمشاهدة البث!
*(ملاحظة: للبث الخارجي الكامل بدون قيود شبكة محلية لـ RTMP، ستحتاج أيضاً لعمل Port Forwarding لمنفذ الـ RTMP 1935 في إعدادات الراوتر الخاص بك).*
